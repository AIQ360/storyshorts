## Installing Dependencies

Once you purchase the code, move into the folder named `source code` and install the necessary dependencies. Make sure you have [Node.js](https://nodejs.org/) (v22 or later) installed on your system.

```bash filename="Terminal" copy
cd "source code"
```

Depending on your package manager, run one of the following commands:

{/* prettier-ignore */}
<Tabs items={["pnpm", "npm", "yarn"]} defaultIndex="0">
  <Tab>
  ```bash filename="Terminal" copy
  pnpm install
  ```
  </Tab>
  <Tab>
  ```bash filename="Terminal" copy
  npm install
  ```
  </Tab>
  <Tab>
  ```bash filename="Terminal" copy
  yarn
  ```
  </Tab>
</Tabs>

<Callout>
  We recommend using **pnpm** as the package manager. It is faster, more
  efficient with disk space, and is what Framecast AI is developed with.
</Callout>
