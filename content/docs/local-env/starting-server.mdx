## Running Framecast AI

If you have followed all of the previous steps correctly, you are ready to run Framecast AI locally. Start the development server with:

{/* prettier-ignore */}
<Tabs items={["pnpm", "npm", "yarn"]}>
  <Tab>
  ```bash filename="Terminal" copy
  pnpm dev
  ```
  </Tab>
  <Tab>
  ```bash filename="Terminal" copy
  npm run dev
  ```
  </Tab>
  <Tab>
  ```bash filename="Terminal" copy
  yarn dev
  ```
  </Tab>
</Tabs>

This will start the Next.js development server. Open [http://localhost:3000](http://localhost:3000) in your browser to see Framecast AI running locally.
