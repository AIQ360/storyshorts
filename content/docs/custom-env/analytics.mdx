## Google Analytics

<Callout type="warn">
  Make sure you already have a [Google Analytics](https://analytics.google.com/)
  account before proceeding.
</Callout>

Framecast AI supports [Google Analytics](https://analytics.google.com/) out of the box. You just need to create a property and add the tracking values to your `.env` file.

<Steps>
  <Step>
    ### Create a Google Analytics Property

    Go to your [Google Analytics](https://analytics.google.com/) dashboard and click the **Settings** icon from the sidebar. Click **Create** and select **Property** from the dropdown. Fill in the required details (Property Name, Business Details, Business Objectives) and click **Create**. Choose **Web** as the platform, enter your website URL and a stream name, then click **Create & Continue**.

    <Screenshot src="/doc-assets/19.png" alt="Google Analytics — Creating Property" />

  </Step>

  <Step>
    ### Get Your Tracking Values

    After creating the property, a setup dialog will appear. Select the **Install Manually** option. You will see two values you need to copy:

    1. The full `src` URL (looks like `https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX`)
    2. The measurement ID (looks like `G-XXXXXXXXXX`)

    <Screenshot src="/doc-assets/20.png" alt="Google Analytics — Tracking ID Setup" />

  </Step>

  <Step>
    ### Update Your .env File

    Open your `.env` file and replace the Google Analytics placeholders with the values you copied:

    ```bash filename=".env" copy
    # Full script URL including https://
    NEXT_PUBLIC_GOOGLE_ANALYTICS_ID=https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX
    NEXT_PUBLIC_GOOGLE_ANALYTICS_ID_VALUE=G-XXXXXXXXXX
    ```

    Replace `G-XXXXXXXXXX` with your actual measurement ID in both lines.

  </Step>
</Steps>

You can verify the integration is working by clicking the **Test** button in the Google Analytics setup dialog. Once confirmed, analytics will start tracking automatically.
